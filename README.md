# Rechiseled Compat (FORK)

**Adds support for various mods to Rechiseled.**

## Versions

### 1.0.x
- Wood blocks

### 1.1.x
- Recipe and tool support

### 1.2.x (WIP)
- Stone blocks from selected mods

## Minecraft Versions

* prior to 1.16.5 (no)

* 1.16.5 (planned)

* 1.18.2 (planned)

* 1.19.2 (out)

* after 1.19.2 (we will see)
