- Ars Nouveau
- Athmospheric
- Atum
- Bayou Blues
- Better End Forge
- Better Nether
- Biomes O' Plenty
- Blue Skies
- Oh The Biomes You'll Go
- Druidcraft
- Endergetic
- Enviromental
- Generic Ecosystem
- Natures Aura
- Outer End
- Quark
- Tinkers Construct
- Traverse
- Tropicraft
- Twilight Forest
- The Undergarden
